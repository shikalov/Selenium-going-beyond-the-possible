Инструкция по установке.
1.	Установить node.js (http://nodejs.org/).
2.	С помощью файлового менеджера(к примеру Far) или в консоли cmd перийти к папке с исходниками (demo) и выполнить команду "nmp i".
3.	Затем "npm i -g bower".
4.	Перейти в каталог public и выполнить команду "bower install".
5.	После вернуться в каталог demo и выполнить команду "node app.js".
6.	В браузере открыть http://localhost:4000.
7.	Пароль и логин к «защишенной странице» user и pass соответственно.
8.	Для остановки сервера необходимо использовать команду Ctrl+C в консоли.
